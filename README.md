# patent-fuzzy-intelligence
Fuzzy logic model for patent portfolio assessment in emerging economies (ICECCME 2026)
Raw patent data is not included due to file size constraints.
To replicate the analysis, download the datasets directly from
Lens.org using the following filters:

- Applicant country: CO, BR, VN, MY
- Publication date: 2020-01-01 to 2025-12-31
- Export fields: Family size, Citations, Jurisdictions, 
  Legal status, CPC classifications

Direct links:
- Colombia: https://www.lens.org (filter: applicant country = CO)
- Brazil: https://www.lens.org (filter: applicant country = BR)
- Vietnam: https://www.lens.org (filter: applicant country = VN)
- Malaysia: https://www.lens.org (filter: applicant country = MY)
